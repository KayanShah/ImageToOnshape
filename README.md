A step-by-step guide to converting an image into a DXF sketch compatible with any CAD software — including Onshape, Fusion 360, SolidWorks, AutoCAD, and FreeCAD. Uses free online tools, no installs required.

> I've been using this method for years to bring reference images into CAD. This repo tidies up and documents the process, with the guide polished using **Claude AI**.

📄 **[Download the full guide (PDF)](https://github.com/KayanShah/ImageToOnshape/blob/main/Onshape_Image_Guide.pdf)**

---

## Pipeline

**Image → Upscale → SVG → DXF → Any CAD Software**

| Step | Action | Tool |
|------|--------|------|
| 1 | Upscale the image | [iLoveIMG](https://www.iloveimg.com/upscale-image) |
| 2 | Convert to SVG | [FreeConvert](https://www.freeconvert.com/svg-converter) |
| 3 | Convert SVG → DXF | [CloudConvert](https://cloudconvert.com/svg-to-dxf) |
| 4 | Import DXF into your CAD software | Built-in to most CAD tools |
